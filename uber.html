Case Study

Uber Trip Data Analysis

From Raw CSV to an Interactive Power BI Dashboard for ride performance and cancellation analytics.

Power BI DAX Power Query
← Back to Portfolio Home

Project Overview

This project involves end-to-end analysis of Uber trip data using Power BI. The goal was to transform raw CSV data into a fully interactive, insight-driven dashboard that highlights ride demand trends, location performance, vehicle comparisons, cancellations, and key business KPIs for decision-making.

Problem Statement

Uber generates a massive volume of trip data daily. The challenge was to clean, model, and visualize this raw data to uncover booking patterns, peak demand time, popular locations, customer & driver cancellations, and vehicle performance in one consolidated analytics solution.

Tools & Technologies

Power BI
Power Query
DAX
Excel / CSV

Project Workflow

1. Raw Data Import

Imported raw CSV files containing trip date, time, pickup locations, distance, fare, payment mode, ride status, and vehicle type.

2. Data Cleaning & Transformation

Removed duplicates, handled missing values, standardized date-time formats, and created derived fields (month, vehicle groups, booking status flags) using Power Query.

3. Data Modeling

Built custom relationships between fact and dimension tables, created calculated columns, and wrote advanced DAX measures for KPIs such as total bookings, conversion rates, cancellation rates, and ride-distance metrics.

4. Visualization & Dashboard Design

Designed interactive report pages for overview, bookings, vehicles, and insights with buttons for page navigation and slicers for dynamic filtering.

Dashboard Pages

Overview Dashboard – total bookings, completed rides, revenue & payment mix.
Insights Dashboard – cancellations, reasons analysis and driver vs customer trends.
Bookings Dashboard – top pickup & drop locations and conversion KPIs.
Vehicle Detail Page – vehicle-wise bookings, distance, ratings & yearly trend.
Raw CSV Data – original trip-level dataset before transformation.
Explore filters and buttons in the live report to navigate between pages.

Key Insights

  • Identified peak booking hours and high-demand months using time-series analysis.
  • Highlighted top pickup and drop locations contributing the highest booking value.
  • Separated customer vs driver cancellations and drilled into top cancellation reasons.
  • Compared performance across vehicle types (Auto, Go Mini, Go Sedan, Premier, eBike, XL) on bookings, revenue and distance.

Business Impact

The dashboard acts as a single source of truth for operations and strategy teams, enabling them to monitor booking performance, customer behaviour, driver reliability and cancellation patterns, and design targeted interventions to improve ride completion and profitability.

Learnings & Skills Gained

  • Advanced Power BI dashboard building and page navigation design.
  • DAX for building reusable measures and business KPIs.
  • Data modeling & transformations using Power Query for analytics-ready tables.
  • Structuring analytics as a clear data story for stakeholders.
